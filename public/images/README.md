## Recommended specs
- **Hero backgrounds**: 1920×1080 px minimum, JPEG 80% quality
- **Content images**: 1200×800 px minimum, JPEG 80% quality
- **Format**: `.jpg` preferred (or `.webp` — update the path in the component accordingly)
